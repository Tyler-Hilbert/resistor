# Resistor
C++ app to identify resistors based off color bands and calculates their resistance in parallel and series.

# Installation 
For Linux:  
$git clone https://github.com/Tyler-Hilbert/resistor.git  
$cd resistor  
$make  
$./resistor  

For Windows:  
$git clone https://github.com/Tyler-Hilbert/resistor.git  
$cd resistor  
$make  
$resistor.exe
